## Requirements

For compiling .d file install Code Runner extension and set:
```javascript
"code-runner.executorMap": {
    "esatan": "C:\\path\\to\\esatan.bat e $dir $fileNameWithoutExt $fileName . no"
    }
```  
in the user settings.json file.
